# Get-WinGetUserSettings

## SYNOPSIS
Gets user settings for WinGet.

## SYNTAX

```
Get-WinGetUserSettings [<CommonParameters>]
```

## DESCRIPTION

Get user settings for WinGet.

## EXAMPLES

### Example 1: Default

```powershell
Get-WinGetUserSettings
```

Gets the WinGet user settings from the user's settings.json file.
